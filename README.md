# 🏋️ BeFit Frontend

Frontend de BeFit, una aplicación web para registrar entrenamientos y progreso físico a lo largo del tiempo.

## ✨ Funcionalidades
- Registro de entrenamientos
- Visualización de progreso
- Autenticación de usuarios

## 🚀 Getting Started

```bash
npm install
npm run dev
```

## 🛠️ Tech Stack
- React
- TypeScript
- Vite
- ESLint + React plugins

## 🔗 Backend
Este frontend consume la API de [BeFit](https://github.com/nailed-mint/befit-backend).

## 📱 Mobile
La aplicación [BeFit](https://github.com/nailed-mint/befit-mobile) también está disponible para dispositivos Android/iOS
